# Technical Summary - Talleres Mañes Web
## Technologies and Methodologies Used (For Europass)

### 1. **Frontend Development with HTML5, CSS3 and Vanilla JavaScript**
   - Creation of a modern multi-section website without frameworks
   - Implementation of semantic HTML5 to improve accessibility and SEO
   - Pure JavaScript (Vanilla JS) for DOM manipulation and interactions

### 2. **Advanced CSS: Glassmorphism and Animations**
   - Visual design with **Glassmorphism** effect (frosted glass) using backdrop-filter
   - Advanced CSS animations (@keyframes) for smooth visual effects
   - Radial and conic gradients for modern visual effects
   - Responsive design system with CSS Grid and Flexbox

### 3. **Interactive JavaScript: Custom Effects**
   - **Interactive flashlight effect** on hero section with dynamic radial gradients
   - Tab navigation system with scroll management
   - **3D tilt effect** on logo with smooth animation using RequestAnimationFrame
   - Accordion for FAQ with interactive toggle
   - WhatsApp API integration for contact forms

### 4. **Responsive Design and Modern UX/UI**
   - Mobile-first adaptive design for mobile devices, tablets and desktop
   - Performance optimization with lazy loading and smooth transitions
   - Intuitive navigation system with scroll-to-section
   - Animated loading screen to improve user experience
   - Floating buttons (WhatsApp, back-to-top) for easy access

### 5. **External APIs and Services Integration**
   - Embedded Google Maps API to display workshop location
   - Google Fonts for custom typography (Montserrat, Rajdhani)
   - WhatsApp Business API integration for message sending
   - Direct links to Google Business reviews

### 6. **Version Control and Project Management**
   - Git and GitHub for code version control
   - GitHub Pages for website deployment and hosting
   - Technical project documentation (README.md)
   - Issues and pending tasks management

## Technical Skills Developed

- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **CSS Techniques**: Glassmorphism, Animations, Gradients, Responsive Design
- **JavaScript**: DOM Manipulation, Event Handling, API Integration, RequestAnimationFrame
- **Tools**: Git, GitHub, GitHub Pages
- **APIs**: Google Maps, Google Fonts, WhatsApp Business
- **Design**: UI/UX, Responsive Web Design, Interactive Visual Effects
