# Resumen Técnico - Talleres Mañes Web
## Tecnologías y Metodologías Utilizadas (Para Europass)

### 1. **Desarrollo Frontend con HTML5, CSS3 y JavaScript Vanilla**
   - Creación de una página web moderna de una sola página (SPA) sin frameworks
   - Implementación de HTML5 semántico para mejorar accesibilidad y SEO
   - JavaScript puro (Vanilla JS) para manipulación del DOM e interacciones

### 2. **CSS Avanzado: Glassmorphism y Animaciones**
   - Diseño visual con efecto **Glassmorphism** (cristal esmerilado) usando backdrop-filter
   - Animaciones CSS avanzadas (@keyframes) para efectos visuales fluidos
   - Gradientes radiales y cónicos para efectos visuales modernos
   - Sistema de diseño responsive con CSS Grid y Flexbox

### 3. **JavaScript Interactivo: Efectos Personalizados**
   - **Efecto linterna interactivo** en el hero con gradientes radiales dinámicos
   - Sistema de navegación por pestañas (tab navigation) con scroll management
   - **Efecto tilt 3D** en el logo con animación suave usando RequestAnimationFrame
   - Acordeón para FAQ con toggle interactivo
   - Integración con WhatsApp API para formularios de contacto

### 4. **Diseño Responsive y UX/UI Moderno**
   - Diseño adaptable (mobile-first) para dispositivos móviles, tablets y desktop
   - Optimización de rendimiento con lazy loading y transiciones suaves
   - Sistema de navegación intuitivo con scroll-to-section
   - Pantalla de carga animada para mejorar la experiencia de usuario
   - Botones flotantes (WhatsApp, back-to-top) para fácil acceso

### 5. **Integración de APIs y Servicios Externos**
   - Google Maps API embebido para mostrar localización del taller
   - Google Fonts para tipografías personalizadas (Montserrat, Rajdhani)
   - Integración con WhatsApp Business API para envío de mensajes
   - Enlaces directos a reseñas de Google Business

### 6. **Control de Versiones y Gestión de Proyectos**
   - Git y GitHub para control de versiones del código
   - GitHub Pages para despliegue y hosting de la página web
   - Documentación técnica del proyecto (README.md)
   - Gestión de issues y tareas pendientes

## Habilidades Técnicas Desarrolladas

- **Lenguajes**: HTML5, CSS3, JavaScript (ES6+)
- **Técnicas CSS**: Glassmorphism, Animaciones, Gradientes, Responsive Design
- **JavaScript**: DOM Manipulation, Event Handling, API Integration, RequestAnimationFrame
- **Herramientas**: Git, GitHub, GitHub Pages
- **APIs**: Google Maps, Google Fonts, WhatsApp Business
- **Diseño**: UI/UX, Responsive Web Design, Efectos Visuales Interactivos
